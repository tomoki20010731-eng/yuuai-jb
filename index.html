ABOUT

人に寄り添う仕事を、
あたたかい職場で。

水度坂友愛ホームは、地域に根ざした介護サービスを行っています。 利用者さま一人ひとりに寄り添いながら、 スタッフ同士も支え合って働ける環境を大切にしています。

未経験でも始めやすい

業務経験は不問です。介護に関心がある方、 人の役に立ちたい方を歓迎しています。

安心して続けやすい

夜勤は安心の2人体制。資格取得支援や各種補助もあり、 長く働きやすい環境づくりを大切にしています。

地域に必要とされる仕事

介護の仕事は、日々の生活と笑顔を支える仕事です。 人とのつながりを感じながら働けます。

LOCATION

2つの拠点で地域を支えています

水度坂友愛ホームは、青谷ケアセンターと寺田ケアセンターの 2拠点で介護サービスを行っています。

青谷ケアセンター

デイサービス友愛 / グループホーム友愛

家庭的な雰囲気の中で、利用者さまに寄り添える拠点です。

寺田ケアセンター

居宅介護支援 / デイサービス / 小規模多機能 / グループホーム

複数のサービスを展開し、幅広い経験を積みやすい拠点です。

Aさん / 介護職

未経験でも安心して始められました

最初は不安もありましたが、先輩が丁寧に教えてくれて安心できました。 利用者さまの笑顔を見ると、この仕事を選んでよかったと感じます。

Bさん / 看護職

人との距離が近いところが魅力です

医療だけでなく、生活全体に関われるところにやりがいがあります。 スタッフ同士の連携も取りやすく、働きやすい職場だと思います。

Cさん / 調理員

食事を通して支えられるのが嬉しいです

毎日の食事を楽しみにしてくださる方がいることがやりがいです。 「おいしかった」のひとことが励みになります。
